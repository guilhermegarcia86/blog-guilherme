Estrutura


package main

import "fmt"

func main(){
    fmt.Printf("Hello word")
}

Os arquivos .go devem iniciar com a definição de pacote (package) que não exige, como em outras linguagens que seja o mesmo nome do arquivo mas é uma boa prática deixar os nomes iguais. Após temos as definiçções de importações com a palavra import que pode ser definada como um bloco quando temos multiplos pacotes importados exemplo:

import (
    "fmt"
    "net"
)

E seguido das funções variáveis e etc.

Visibilidade

A visibilidade em GO é definida por convenções, como muitas coisas nessa linguagem, e para visibilidade de structs, functions, variaveis, contantes e etc é usado a seguinte convenção, tudo o que começar com letra maiúscula será público e tudo o que começar com letra minúscula será privado, exemplo:

func functionprivada(){
// função privada
}

func FunctionPublica(){
// função pública
}

Variáveis e Constantes

Podemos declarar as variáveis com a palavra var seguida do tipo e seu valor:

var num int32 = 42

E contantes da mesma forma com a palavra reservada const:

const PI float64 = 3.1415

Temos também a opção de iniciarmos blocos de variávies e constantes:

var (
    a = 3
    b = 4
)
const (
    c = 5
    d = 6
)

Apesar das duas abordagens serem simples o mais comum é não tipar as variáveis e constantes já que o GO nos provê inferência de tipos automaticamente:

num := 42

Com operador ":=" é o que faz a inferência de tipos automáticamente, isso que dizer que o tipo é definido de forma automática. Porém como o GO é uma linguagem fortemente tipada não podemos mudar o tipo de variável após a sua declaração.

Lógicos


resultI := true
resultII := false

AND => resultI && resultII = false
OR => resultI || resultII = true
NOT => !(resultI && resultII) = true

If/Else

value := 2

if value > 3 {
    fmt.Println("Valor maior que 3")
} else {
    fmt.Println("Valor menor que 3")
}

// If/Else aninhado
if value > 4 {
    fmt.Println("Valor maior que 4")
} else if value > 3 {
    fmt.Println("Valor maior que 3")
} else {
    fmt.Println("Valor menor que 3")
}
